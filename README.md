# README

## 介绍
欢迎您访问openAN官方文档仓库，参与openAN开发者文档开源项目，与我们一起完善开发者文档。
openAN docs仓包含了快速入门（包含项目描述、软件安装指南、快速入门操作）、安全技术白皮书、RN（Release Notes）文档。

## OpenAN文档地址
- 快速入门、安全技术白皮书、RN：https://gitcode.com/OpenAN/docs
- 注册中心用户指南、开发指南：https://gitcode.com/OpenAN/registry-center/tree/main/docs
- 编排中心用户指南、开发指南：https://gitcode.com/OpenAN/orchestration-center/tree/main/docs
- A2A-T Python SDK用户指南、开发指南：https://gitcode.com/OpenAN/a2a-t-sdk/tree/main/docs
- A2A-T Java SDK用户指南、开发指南：https://gitcode.com/OpenAN/a2a-t-java/tree/main/docs

## 参与贡献
您可以对现有文档进行评价、反馈文档问题、贡献您的原创内容。可以参考以下贡献方式：
- 建立或回复issue：欢迎通过建立或回复issue参与讨论和贡献。
- 提交PR：欢迎通过提交PR的方式参与贡献。
- 提交评论：欢迎在issue或PR中提交评论。
